# FDPACK: Finite Difference Package with Adaptive Offset Selection

FDPACK is a Fortran library for numerical differentiation of 
multivariate functions.
It provides adaptive offset selection for gradient and Hessian 
computations via forward and central finite difference schemes.

## Documentation

The `doc/` directory contains the following files:

- `SUMMARY.pdf` — Detailed description of the underlying algorithms
- `INSTALL.md`  — Installation and compilation instructions
- `TESTS.md`    — Description of the test suite and how to run it
